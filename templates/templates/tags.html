Теги в django

autoescape

Управление текущим экранированием
{% autoescape on %} {{ html_code }} {% endautoescape %} {% autoescape off %} {{ html_code }} {% endautoescape %}

load

Загружает настраиваемый набор тегов шаблона

extends

Сигнализирует, что этот шаблон расширяет родительский шаблон

block

Определяет болок, который может быть переопределен дочерними шаблонами

include

Загружаеь щаблон и отображает его с текущим контекстом

comment

Блок комментариев
{% comment %}

qeafad fwefwef wefwef ewfwef

{% endcomment %}

csrf_token

Тег для защити csrf
{% csrf_token %}

if

Логический оператор
    {% if var1 == None %}

    var1 равно None

    {% endif %}

for

Перебор
    {% for item in some_list %}
  • {{ item }}
    • {% endfor %}

for...empty

for...empty заменяет комбинацию if и for для проверки списка на наличие значений
{% for item in some_list %}
  • {{ item }}
    • {% empty %}
  • Нет новых данных((((
    • {% endfor %}

    cycle

    Циклично выводит элементы из списка
    {% for item in some_list %} {% endfor %}
    {{ item }}

    ifchanged

    Проверяет, изменилось ли значение с последней итерации цикла
    {% for item in some_list %} {% ifchanged %} {% endifchanged %} {% endfor %}
    {{ item }}

    filter

    Фильтрует содержимое блока
    {% filter force_escape|lower %} This text will be HTML-escaped, and will appear in all lowercase. {% endfilter %}

    firstof

    Выводит первую переменную аргумента, котороя не является ложной
    {% firstof var1 var2 var3 var4 var5 %}

    now

    Отображет текущую дату и время в соответствие с форматом
    {% now 'jS F Y H:i' %}
    {% now 'DATETIME_FORMAT' %}

    spaceless

    Удаляет проблемы между тегами HTML
    {% spaceless %}

    Foo

    {% endspaceless %}

    templatetag

    Выводит один из синтаксических символов
    {# {% templatetag openblock %}#}

    url

    возвращает ссылку на абсолютный путь
    Фильтры

    with

    кэширует сложную переменную под простым именем
    {% with query = obj.name %} {{ query }} {% endwith %}

    lorem

    Отображает текст lorem ipsum
    {% lorem 2 p random %}

    debug

    выводит отладочную информацию
    {% debug %}
